RocketLearn

RocketLearn is an app which allows users to study vocabulary in an authentic, engaging and entertaining way while retaining control of the experience.

My Process

Research

I started the process by conducting user research with users of similar apps and services. Through this I discovered that users:

  • Want a time efficient way to study - preferably in small, bite sized sessions
  • Would be more likely to return to the app if the experience was fun and rewarding
  • Want to be able to personalise their learning experience
  • Want to be held accountable without feeling pressured

Ideation

Based on my research I created a protopersona to ensure that I kept focus on the user throughout the process. This also helped me develop the problem statement which I ultimately wanted Rocketlearn to solve. Through task analysis and user flows I developed a structure for the app which I would deliver an efficient and user-friendly experience.

Wireframing & Testing

I created a series of low-fidelity wireframes which I used to build a functioning prototype in Prott. I then recruited 5 participants to take part in in-person user tests of the app. The issues and how I decided to resolve them were as follows:

  • Added a ‘HELP’ button to menu in order to access support at all times
  • The user wasn’t able to confirm when setting the time for a challenge so a button for this was added
  • Some users were frustrated about being able to use the app right away so an option to skip onboarding was included
  • Naming of some elements caused some confusion and was therefore clarified in the next iteration
